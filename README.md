## Описание

Это учебный проект, разработанный для практического обучения программированию.

## Требования

- Python 3.8 или выше
- pip (менеджер пакетов Python)
- Необходимые зависимости (см. requirements.txt)

## Установка

1. Клонируйте репозиторий:
```bash
git clone <url-репозитория>
cd praktika_GaliullinRustam34410A
```

2. Создайте виртуальное окружение:
```bash
python -m venv .venv
```

3. Активируйте виртуальное окружение:
   - На Windows:
   ```bash
   .venv\Scripts\activate
   ```
   - На Linux/Mac:
   ```bash
   source venv/bin/activate
   ```

4. Установите зависимости:
```bash
pip install django
```

## Запуск

Для запуска приложения выполните:
```bash
cd .\project
python manage.py makemigartions
python manage.py migrate
python manage.py runserver
```

## Использование

1. Запустите приложение согласно инструкциям выше
2. Следуйте инструкциям на экране
3. Введите необходимые данные
4. Получите результаты работы программы

## 📖 Использование

```
Через веб-интерфейс
1. Откройте http://127.0.0.1:8000/
2. Заполните форму «Новая заявка»:
    - Описание — текст проблемы клиента
    - Статус — выберите из списка (Новая / В работе / Закрыта)
3. Нажмите «Добавить заявку»
4. Заявка появится в таблице ниже
Управление заявками:
 - Фильтры — нажмите на «Все», «Новая», «В работе» или «Закрыта» для фильтрации
 - В работу — переводит заявку из статуса «Новая» в «В работе»
 - Закрыть — переводит заявку из статуса «В работе» в «Закрыта»
 - Удалить — удаляет заявку (с подтверждением)
```
